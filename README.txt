Platforms Used:
Visual Studio Code - For coding the dashboard and Node JS server.
Arduino IDE - For coding the Arduino code and water height calculations.
Node JS - For reading data from the serial port and outputting it to the dashboard, as well as allowing the mobile app to connect.
Plotly.js - For displaying the data as graphs.
Microsoft PowerPoint - For making the presentation.
Google Docs - For making our plan and a place to put our research.
Github - For uploading our code to the cloud.
Xcode - For coding our iOS app.
Code HS - For testing elements of the dashboard before implementing them into our main code.
